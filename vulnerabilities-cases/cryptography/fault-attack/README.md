# 故障攻击

## 定义

在硬件工业中，**故障攻击**指的是对集成电路或其状态的恶意操纵，从而引起电路错误而将设备引入一个不正常的状态。其目标是访问关键安全信息或停止内部的保护机制。&#x20;

在区块链或软件领域，我们同样把有类似攻击路径和结果的行为称为故障攻击。
