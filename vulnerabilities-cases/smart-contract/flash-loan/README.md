# 闪电贷

使用闪电贷需要区块链开发相关知识，是为开发者设计的。闪电贷可以在没有任何抵押品的情况下进行借贷，只要借贷的流动性在同一交易中返还给资金池即可。调用闪电贷需要写一个智能合约，合约中的会声明所有的使用步骤，最后将贷款和利率在同一笔交易中返还。
